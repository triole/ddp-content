**DDP-Bildung**

**Anwendungsfall II.a):**

***Dokumentation der Untersuchungspopulation\
bei Querschnittsdaten***


Ausgangsbeispiel:
=================

Ein Forschungsprojekt untersucht schulische Leistungen während der
dualen Berufsausbildung im Kontext beruflicher Belastungen am
Arbeitsplatz. Dazu werden insgesamt 1000 Berufsschüler\*innen in
Rheinland-Pfalz mittels eines standardisierten Fragebogens befragt.

Beispiel zur Dokumentation der Untersuchungspopulation
======================================================

+---------------------+-----------------------------------------------+
| geographischen Raum | Rheinland-Pfalz, Deutschland (DE-RP)          |
+=====================+===============================================+
| Grundgesamtheit     | Schüler\*innen an Berufsschulen               |
+---------------------+-----------------------------------------------+
| Auswahlverfahren    | zweistufige Zufallsauswahl                    |
|                     |                                               |
|                     | 1.  auf Basis aller Berufsschulen in          |
|                     |     Rheinland-Pfalz werden zufällig zehn      |
|                     |     Schulen ausgewählt                        |
|                     |                                               |
|                     | 2.  auf Basis des                             |
|                     |     Schüler\*innen-Verzeichnisses der         |
|                     |     ausgewählten Berufsschule werden zufällig |
|                     |     100 Schüler\*innen ausgewählt             |

+---------------------+-----------------------------------------------+
